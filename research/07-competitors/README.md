# Competitor Analysis

## Landscape Overview

IndraNet competes in two adjacent markets:
1. **GPU cloud marketplaces** (Vast.ai, RunPod) — headless GPU rental
2. **Cloud gaming / remote desktop** (Shadow PC, GeForce NOW, Parsec) — interactive streaming

IndraNet's unique position: **interactive streaming + open hardware marketplace**.
Neither camp does both.
